Function of this program:
 1. Download all videos on a Youtube playlist as .mp4 files.
 2. Creates new files from all files that end with .mp4 in selected folder.
 3. DELETES ALL files in folder that ends with .mp4 for data management.

What you need:
 1. Create a new folder.
 2. Copy its file path and paste it into destination keyword/variable.
 3. Copy your link to your Youtube play list and paste it into my_playlist
 key word/variable.
 4. Run code, confirm the final step due to data deletion
 5. Enjoy your music in devices that can only run .mp3 format.

 Note:
 Cannot download age restricted videos.
